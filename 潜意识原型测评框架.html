02 目标受众

谁会做这个测评

测评设计覆盖两类用户，核心钩子需要同时打动两类人。

暖用户——已经对心理成长、催眠、潜意识感兴趣，正在寻找一个系统方法的人。他们的核心问题是「为什么我学了很多但改变很有限」。
冷流量——完全不了解催眠，但正在经历关系困境、目标受阻、情绪内耗的人。他们的核心问题是「我已经很努力了，为什么还是过不好」。
心理咨询师 / 教练——有助人动力，想扩展专业工具库的从业者。他们的核心问题是「我需要一个更底层、更直接的改变方法」。
家长 / 教育工作者——关注亲子关系、代际模式，想打破家庭能量循环的人。他们的核心问题是「我不想把自己的模式传给下一代」。
03 理论框架

测评测什么

基于吉利根生生不息催眠体系，提取三个核心测量维度。三个维度是递进关系，不是并列关系。

测评的底层逻辑来自催眠的核心发现：每个人天生携带三种原型能量，但在成长过程中，这些能量可能被压制、扭曲，或锁结在身体里无法流动。测评帮用户看见自己的能量状态，报告里的模型图会解释这条逻辑链。

维度一 原型能量构成
「你携带了哪些能量，哪些是充足的，哪些是受限的？」
测量温柔、勇猛、顽皮三种能量各自的发展程度。三种能量的高低组合，决定用户属于8种原型类型中的哪一种。
维度二 原型动力
「你的能量是自然流动出来的，还是靠紧绷驱动的？」
测量能量外化的方式。有能量，不代表能用出来——用的方式，决定了你是在创造还是在消耗。这正是报告模块一模型图中「COACH创造性流动」与「CRASH神经肌肉锁结」的测量依据。
气脉顺畅 · 创造性流动 COACH vs 肌肉锁结 · 神经肌肉锁结 CRASH
维度三 原型整合
「你能不能在不同情境下，把三种能量都活出来？」
测量三种能量能否灵活切换和整合。有能量、能用出来，不代表能整合——整合是三个维度里最高层次的能力。
三个维度的递进关系
维度一：你有什么能量　→　维度二：你怎么用这些能量　→　维度三：你能不能把它们整合在一起

MODULE 01 你携带的能量
维度一 · 唯一输入
模块一承担三层叙事，在用户进入报告的第一时间同时完成三件事：

第一层 · 原型类型画像
基于三种能量的得分组合，呈现用户的原型类型（8种之一）。用有温度的语言描述「你是这样一个人」——先命中用户最核心的体验，再描述力量所在，最后呈现核心张力。先肯定，再呈现张力，不是「你有问题」，是「这里是你的力量，这里是你还没打开的门」。

第二层 · 原型能量是什么
一句话建立概念：温柔、勇猛、顽皮是每个人与生俱来的三种原型能量，它们本来是自然流动的。大多数人和它们的关系，都在成长过程中被改写了。

第三层 · 催眠和能量的关系（模型图）
用可视化模型建立认知：能量为什么会被锁住，锁在哪里，为什么头脑层面的努力触碰不到，催眠如何解锁。催眠在这里第一次出现，不是作为课程推销，而是作为理解自己的框架工具。

MODULE 02 你形成的模式
维度一细节维度二
两层递进内容，有明确的衔接逻辑：

第一层 · 原生家庭模板（心理层面）
原生家庭如何塑造了你与三种能量的关系模板。哪些能量被允许，哪些被压制，哪些被扭曲。这个模板当年保护了你，但现在让你付出了代价。

衔接句：「这个模式不只住在你的头脑里，它住在你的身体里。」

第二层 · 身体运转方式（身体层面）
用非常日常的语言描述锁结状态：开口前不自觉屏气、越努力越耗竭、想放松却放不下来。这不是性格，这是神经系统学会的保护模式。
情感钩子 + 核心钩子
「你已经知道了——努力没有用，道理没有用，因为锁住你的不是你的头脑，是你的身体。而能直接和身体对话的方法，不是分析，不是意志力，是催眠。」
→ 回答「为什么是催眠而不是其他方法」这个所有用户都会有的隐性问题
MODULE 03 你被卡住的地方
维度一 × 维度二 · 交叉推论
把用户的原型能量状态映射到五个生命层次，每层同时呈现两件事：你在哪里顺畅显化（天赋入口），你在哪里卡住（成长缺口）。

每层对应一个课程阶段，钩子有双版本：卡点型（解决痛苦）和显化型（放大天赋），覆盖两类不同动力的用户。

MODULE 04 你的英雄之旅
三个模块各取核心结论 · 综合输入
三层递进，完成从触动到行动的最后转化：

第一层 · 肯定你的起点
有显化层：肯定用户在顺畅显化层已有的真实力量。无显化层（五层全卡）：肯定用户愿意做这个测评本身——「这就是你内在那股推你向前的能量，它是真实的」。

第二层 · 你的专属成长路径
基于三个模块的核心结论，组合出一段话：你已有的力量 + 你身体现在的状态 + 你最值得先打通的层 = 你的专属路径。

第三层 · 一个立刻能用的小练习
与用户的主导原型类型对应，30秒能完成，必须是身体层面的感知而不是头脑层面的思考。让用户在报告结束前，亲身体验一次「原型能量是真实可感知的」。
行动钩子
「你的潜意识已经在这份报告里留下了线索。下一步，是找一个懂它的人，帮你把它打开。」
